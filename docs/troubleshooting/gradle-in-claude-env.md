# Gradle не запускается в среде Claude (ассистент не может собрать мод)

## Симптом

- Любая `.\gradlew.bat ...`, запущенная **ассистентом**, падает на форке процесса
  (loopback / socket).
- Ассистент не может выполнить полноценную сборку NeoForge у себя.

## Причина

Среда агента блокирует loopback-сокет, нужный Gradle-даемону. Полноценная сборка мода
(NeoForge / Minecraft) в ней невозможна.

## Решение — двухконтурная проверка

1. **Чистая Java** (пакеты `domain`, `application`, включая `application.port` — всё без Minecraft)
   проверяется **автономным харнессом**: `javac --release 21` + кэшированный JUnit 5.11.4 из
   `~/.gradle/caches/`. Скрипты лежат в scratchpad (`run_pure_tests.sh`, `SyntTestRunner.java`).
   Для чистого кода это эквивалент Gradle-тестов.
2. **Код Minecraft** (`content`, `network`, `bootstrap`-рантайм, `registry`, `client`, `gametest`)
   собирает и запускает **пользователь** у себя:
   ```
   .\gradlew.bat compileJava testClasses runGameTestServer --no-daemon --no-watch-fs --max-workers=1
   ```

## Ограничение: loopback-тесты тоже не работают у ассистента

Средой запрещены и локальные сокеты (`127.0.0.1`): юнит-тесты, поднимающие мок-HTTP-сервер
(например, `OllamaLlmBackendTest`), в харнессе падают со стеком
`sun.nio.ch.PipeImpl$Initializer$LoopbackConnector`. Это не ошибка кода. Такие тесты:

1. компилируются харнессом (это проверка типов/API),
2. исключаются из локального прогона ассистента (`SyntTestRunner` выбирает пакеты
   `ai.backend`/`ai.orchestration`, но не `ai.ollama`),
3. выполняются у пользователя: `.\gradlew.bat test` (или полный `build`).

## Как делить работу

- Каждый срез режется на: **чистую часть** (проверяется харнессом → коммит сразу) и
  **Minecraft-часть** (сборка у пользователя → коммит после «всё зелёное»).
- Интерфейсы-порты (`application.port`) считаются чистыми и компилируются харнессом,
  но их **реализации** в `content/*` проверяет только сборка пользователя. Поэтому добавление
  метода в порт кладите в тот же кусок, что и реализацию, — иначе сборка пользователя временно
  сломается.
